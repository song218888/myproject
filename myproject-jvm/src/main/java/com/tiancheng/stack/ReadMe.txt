1：栈里存放着什么，存放着一个个的栈桢，一个方法被调用时，就会创建一个栈桢
2：栈桢里又存放着什么呢：
	方法内变量，即临时变量
	方法参数
	引用变量
	中间结果
	基础数据类型
	
	
3：什么是栈的深度：
	每个方法在执行的同时都会创建一个栈帧，栈里允许创建的多少个栈桢，叫栈的深度



4：堆里存放着什么
	对象数据
	
5：运行时区域存放着什么
	I：常量，静态变量
	II：类相关信息，如类名，类的修饰符，方法名
	III：用持久代来存储
