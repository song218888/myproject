1：简单工厂模式
   I：工厂是用来生产不同的产口的
   II：通过接口定义产品
   III：不同的产品实现共同实现相同的接口
   IV：利用工厂的方法，根据不同的入参值，new出不同的产品
   V：简单工厂的不足在于，每次新增加产品的时候，工厂方法都需要改变，违反了设计模式的“开闭原则”
   VI：工厂模式主要对产品抽象

2：工厂方法模式
   I：故名思意，就是工厂模式的方法进行接口
   II：将原本简单工厂的入参方法功能，抽成一个单独的接口，如：IFactory接口，定义构造工厂的方法
   III：实现工厂接口，拥有不同工厂接口的实现类
   IV：工厂方法主要对工厂抽象
